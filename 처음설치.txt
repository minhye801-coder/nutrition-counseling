nutrition-counseling 설치 순서
===============================

이 프로젝트는 Cloudflare Pages가 아니라 Cloudflare Workers 전용입니다.

1. GitHub에 새 저장소 생성
   저장소 이름: nutrition-counseling

2. 이 ZIP 압축을 푼 뒤 아래 항목을 저장소 최상단에 업로드
   - public
   - src
   - package.json
   - wrangler.jsonc
   - README.md
   - .gitignore
   - .dev.vars.example

3. Cloudflare에서
   Workers & Pages
   → Create
   → Import a repository
   → nutrition-counseling 선택

4. 배포 설정
   Project name: nutrition-counseling
   Production branch: main
   Build command: 비워 둠
   Deploy command: npx wrangler deploy
   Root directory: 비워 둠

5. 정상 주소 확인
   https://nutrition-counseling.<계정명>.workers.dev

   아래 주소는 사용하지 않음
   https://nutrition-counseling.pages.dev

6. Worker 설정
   Settings
   → Variables and Secrets
   → Add

   Type: Secret
   Variable name: NEIS_API_KEY
   Value: 나이스 인증키

7. 저장 후 재배포하고 학교 검색 시험

중요
----
- NEIS_API_KEY를 GitHub 코드에 직접 넣지 않습니다.
- GitHub 최상단에 public, src, wrangler.jsonc가 바로 보여야 합니다.
- ZIP 파일 자체를 GitHub에 올리는 것이 아니라 압축을 풀어 파일들을 올립니다.
